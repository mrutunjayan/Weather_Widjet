# 🌦️ Weather Widget App

A simple and responsive Weather App built using React.  
It fetches real-time weather data using OpenWeather API.

---

## 🚀 Features

- 🔍 Search weather by city name
- 🌡️ Shows temperature, min & max temp
- 💧 Displays humidity
- 🤒 Feels like temperature
- ☁️ Weather description (cloudy, sunny, etc.)
- ❌ Error handling for invalid city

---

## 🛠️ Tech Stack

- React.js
- Material UI
- OpenWeather API
- JavaScript (ES6)
